# min-webapp

Minimum boilerplate of isomorphic web application

## Features

* React with Server-Side Rendering (SSR)
* CSS modules

## Workflow

* Build production

    ```
    npm run build
    ```

* Start app

    ```
    npm start

    ```

* Watch file changes and incremental build with running development server

    ```
    npm run watch
    ```
